Проекты

На данной странице отображены все продукты, которые использовались для разработки и деплоя данного сайта, а так же некоторые вспомогательные инструменты

GitLab

Для внесения изменений и контроля версий была выбрана самая популярная система

Jenkins

Для автоматического деплоя сайта в докер контейнер при внесении изменений, была выбрана эта система

Docker

В качестве системы контейнеризации был выбран самый популярный и удобный продукт

Zabbix

Для мониторинга системы работает Zabbix, отслеживает состояние сервера, присылает алерты

HTML/PHP/CSS/JS

Привычный стек программ для создания сайта был использован и в данном случае

ELK

Стек elasticsearch, logstash, kibana для сбора и хранения логов
